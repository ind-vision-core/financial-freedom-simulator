A sophisticated financial simulator that computes 1,000 parallel realities based on 100 years of real market mathematics. This interactive time machine helps users explore investment horizons, compound interest, and the statistical paths to financial independence.

## 🚀 Features
- **1,000 Parallel Realities:** Simulates diverse financial outcomes using clean mathematical models inspired by historical market data.
- **Premium UI:** A sleek, modern, dark-themed interface built for data visualization.
- **Interactive Charts:** Real-time data visualization driven by Chart.js to track median and extreme financial outcomes.
- **Web3 Integrated:** Features a minimalist, built-in crypto tip box supporting MetaMask (BEP20 / BSC network).
- **100% Client-Side:** Lightweight, secure, and fast. No database or server-side overhead required.

## 🛠️ Tech Stack
- **Structure:** HTML5
- **Styling:** Tailwind CSS
- **Charts:** Chart.js
- **Logic:** Vanilla JavaScript (ES6+)
